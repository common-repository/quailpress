QuailPress allows you to intergrate with Facebook Share so that users can easily share your posts on their Facebook accounts.
== Installation ==

1. Extract plugin archive into '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress3. Place `<?php quailpress_share(); ?>` in your templates== Screenshots ==
